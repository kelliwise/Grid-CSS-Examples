These are examples of CSS Grid, layouts, CSS and how it can be used.

The basic Grid

Using shorthand

.wrapper1 .a {
        grid-column: 1 / 3;
        grid-row: 1;    }
.wrapper1 .b {
        grid-column: 3 ;
        grid-row: 1 / 3;    }
.wrapper1 .c {
        grid-column: 1 ;
        grid-row: 2 ;    }
.wrapper1 .d {
        grid-column: 2;
        grid-row: 2;    }
.wrapper1 .e {
        grid-column: 1/3;
        grid-row: 3;    }
.wrapper1 .f {
        grid-column: 3;
        grid-row: 3;    }

Variation of the basic Grid

Here we use spans

.wrapper3 .a {
        grid-column-start: 1;
        grid-column-end:span 2;
        grid-row: 1;    }
.wrapper3 .b {
        grid-column: 3 ;
        grid-row-start: 1;
        grid-row-end: span 2;    }
.wrapper3 .c {
        grid-column: 1 ;
        grid-row: 2 ;    }
.wrapper3 .d {
        grid-column: 2;
        grid-row: 2;    }
.wrapper3 .e { 
        grid-column-start: 1;
        grid-column-end:span 2;
        grid-row: 3;    }
.wrapper3 .f {
        grid-column: 3;
        grid-row: 3;    }

Variation of the basic Grid

We're going to move things around and stretch them out. Again, the html of these, except for div classes, are identical to the previous examples.

Here, block B is on the right.

.wrapper2 .a {
        grid-column: 1 / 4;
        grid-row: 1;    }
.wrapper2 .b {
        grid-column: 3 ;
        grid-row: 2/4;
        background-color: red;    }
.wrapper2 .c {
        grid-column: 1 ;
        grid-row: 2 ;    }
.wrapper2 .d {
        grid-column: 2;
        grid-row: 2;    }
.wrapper2 .e {
        grid-column: 1/3;
        grid-row: 3;    }
.wrapper2 .f {
        grid-column: 1/4;
